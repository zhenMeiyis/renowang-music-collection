# renowang-music-collection
王铮亮的音乐合集网页demo版

## 简介
王铮亮音乐合集展示网站，爬取并整理了**小亮在QQ音乐截至2026年2月11日的全部歌曲信息**，目前共收录623条记录（含伴奏、live等版本），做了可视化的网页展示，支持搜索、筛选、排序、查看歌曲详情和歌词等功能～ 同时也蹲蹲感兴趣的宝子一起建设，补全信息、分类音乐类型都超欢迎✨

注：本项目仅供学习交流使用，所有歌曲版权归原著作权人所有。

## 项目亮点
✅ 收录Q音全量歌曲：622条记录，含歌曲ID、名、歌手、专辑、发行日期、作词作曲、歌词等多字段  
✅ 可视化网页浏览：蓝白现代风设计，适配电脑/手机，操作简单直观  
✅ 多功能检索：搜索+多维度筛选+排序，快速找到想听的歌  
✅ 沉浸式详情页：查看歌曲完整信息，跳转Q音播放，同步展示歌词  

## 本地运行方法
### 方法一
直接打开网址：https://zhenmeiyis.github.io/renowang-music-collection/
不过这个很看运气，有时候可能进不来，进不来的话可以考虑方法二
如果对程序一窍不通的话可以看excel版：

### 方法二
超简单！只要电脑装了Python，几步就能跑起来～
1. 克隆仓库到本地
```bash
git clone https://github.com/你的用户名/renowang-music-collection.git
cd renowang-music-collection
```
2. 在项目根目录依次执行两条命令
```bash
python auto_run.py
python -m http.server 8000
```
3. 打开浏览器，输入地址就能访问啦
```
http://localhost:8000
```

## 网站使用指南
### 🔍 搜索歌曲
- 顶部搜索框输入**歌曲名/歌手名/专辑名**，回车或点搜索按钮就能搜，关键词会自动高亮
- 搜错了点旁边的清空按钮，一键恢复全部歌曲
- 搜索框点一下会显示历史记录，点历史标签快速重搜，也能一键清空历史

### 📌 筛选歌曲
- **演唱形式**：直接点独唱/合唱等按钮，快速筛对应类型
- **发行年份**：下拉选具体年份，也能选「未知时间」筛无有效发行时间的歌曲
- **专辑筛选**：下拉选专辑名，直接看该专辑下的所有歌

### ⬆️ 排序歌曲
在筛选栏右侧点排序按钮就行，两种排序方式任你选：
- 按发行时间：最新发布/最早发布
- 按歌曲名：A-Z拼音排序/Z-A反向排序

### 🎶 查看歌曲详情
随便点一个歌曲卡片，就能进入详情页：
- 看全量信息：歌手、专辑、发行时间、演唱形式、歌曲ID、QQ音乐直达链接
- 看歌词：居中展示，滚动就能浏览全部歌词
- 想回列表？点左上角「返回列表」按钮就好啦
